# File: src/components/BacktestForm/BasicInfoSection.jsx

## ELI5: What is this file?
This is **Part 2 of the Form**. It asks for the names: "What do you want to call this test?" and "Which stock symbol are we testing?" (like AAPL or BTC).

## How it works
- It draws two simple text inputs.
- When you type, it calls `handleChange` to save your text into the `formData`.

## Summary for Developers
- Pure presentation component. No complex logic.
